# TODO_APP
This is a TODO app developed using Django framework. It handles basic CRUD operations with respect to the TODO tasks such as creating a task, deleting a task, updating, and for views.
<hr/>


To run this project, clone using: 
- git clone https://github.com/mohithnovoct/TODO_APP.git


After cloning the project, run this command:
- pip install -r requirements.txt



After the installations of the necessary packages run the server:
- python manage.py runserver
